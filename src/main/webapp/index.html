Examples of using mappers

Two kind of simple mappers can be found as a REST services:

  • CPV 2008 mapper that takes as a parameter a label and returns 3 mappings.

    • The URL to query the REST the service is: %APP_URL%/services/mapping/cpv2008?label=param
    • Local example: Search for "Services"
    • Remote example: http://pscs-portal.herokuapp.com/services/mapping/cpv2008?label=Services

  • Australia suppliers mapper that takes as a parameter a label and returns 1 mapping (it only uses existing supplier names due to the process to create the mappings has been made off-line using Pyton).

    • The URL to query the REST the service is: %APP_URL%/services/mapping/supplier?label=param
    • Local example: Search for "Oracle Australia Pty Limited"
    • Remote example: http://pscs-portal.herokuapp.com/services/mapping/supplier?label=Oracle Australia Pty Limited

Resources

  • WAR application to be deployed locally (26MB)
  • Java REST test client (Maven project) with two Junit tests to query the REST service (3,8 kB)
